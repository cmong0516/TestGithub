까먹어서 기록하는 Window 단축키!!

*setting - keymap에서 단축키 확인 가능

1. setting 열기
ctrl+alt+s

2. 파일 만들기
ctrl+alt+ins

3. 생성자, 게터 세터 만들기
alt+ins

4. 빨간줄(에러) 잡기
ctrl+1

5. 최근에 연 파일 찾기 -> 최근순으로 파일 목록보여줌
ctrl+e

6. 변수 생성 -> 김영한님 강의에 많이 나오는 단축키
ctrl+alt+v
ex) new Repository(); --> Repository repo = new Repository();

7. ctrl+space(코드힌트)사용 시 끝에 세미콜론까지 붙이기
ctrl+shift+enter

8. extract method
alt+shift+m

9. 함수 단축키
    1) sout 기본 println
    2) soutm 가까운거 출력해주는거같아요 - 해당 메서드 명 출력
    3) soutv 골라서 출력가능 - 변수명 출력
    4) iter - 자동 (향상된)포문 (리스트나 배열이 있을 시에만)
